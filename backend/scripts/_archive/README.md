# Archived Script Suites

This directory stores one-off or legacy testing utilities that no longer ship with the active scripts:

- 	esting_suite/ – historical ad-hoc backend test harnesses.
- manual_tests/ – Puppeteer and exploratory flows used during early UI development.
- 	est_api_providers/ – experimental coverage scripts for third-party provider integration.

Nothing in this folder is part of the supported workflow. Refer to ackend/scripts/README.md for the current script layout.
